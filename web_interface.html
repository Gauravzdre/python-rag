üí¨ Chat Endpoints

POST /query

Send a query to the RAG system

curl -X POST "YOUR_DOMAIN/query" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "What is the main topic of the document?"}'

POST /query/tenant

Send a tenant-specific query

curl -X POST "YOUR_DOMAIN/query/tenant" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "What is the main topic?", "tenant_id": "your_tenant_id"}'

üìÑ Document Endpoints

POST /upload

Upload a document to the default tenant

curl -X POST "YOUR_DOMAIN/upload" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf"

POST /upload/tenant

Upload a document to a specific tenant

curl -X POST "YOUR_DOMAIN/upload/tenant?tenant_id=your_tenant_id" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf"

GET /tenants/{tenant_id}/documents

Get all documents for a tenant

curl -X GET "YOUR_DOMAIN/tenants/your_tenant_id/documents" \
  -H "Authorization: Bearer YOUR_API_KEY"

üè¢ Tenant Management

GET /tenants/

List all tenants

curl -X GET "YOUR_DOMAIN/tenants/" \
  -H "Authorization: Bearer YOUR_API_KEY"

GET /tenants/{tenant_id}

Get specific tenant information

curl -X GET "YOUR_DOMAIN/tenants/your_tenant_id" \
  -H "Authorization: Bearer YOUR_API_KEY"

GET /collection-info

Get collection information for current tenant

curl -X GET "YOUR_DOMAIN/collection-info?tenant_id=your_tenant_id" \
  -H "Authorization: Bearer YOUR_API_KEY"
