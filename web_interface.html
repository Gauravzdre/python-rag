💬 Chat Endpoints

POST /query

Send a query to the RAG system

curl -X POST "http://localhost:8000/query" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "What is the main topic of the document?"}'

POST /query/tenant

Send a tenant-specific query

curl -X POST "http://localhost:8000/query/tenant" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "What is the main topic?", "tenant_id": "your_tenant_id"}'

📄 Document Endpoints

POST /upload

Upload a document to the default tenant

curl -X POST "http://localhost:8000/upload" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf"

POST /upload/tenant

Upload a document to a specific tenant

curl -X POST "http://localhost:8000/upload/tenant?tenant_id=your_tenant_id" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf"

GET /tenants/{tenant_id}/documents

Get all documents for a tenant

curl -X GET "http://localhost:8000/tenants/your_tenant_id/documents" \
  -H "Authorization: Bearer YOUR_API_KEY"

🏢 Tenant Management

GET /tenants/

List all tenants

curl -X GET "http://localhost:8000/tenants/" \
  -H "Authorization: Bearer YOUR_API_KEY"

GET /tenants/{tenant_id}

Get specific tenant information

curl -X GET "http://localhost:8000/tenants/your_tenant_id" \
  -H "Authorization: Bearer YOUR_API_KEY"

GET /collection-info

Get collection information for current tenant

curl -X GET "http://localhost:8000/collection-info?tenant_id=your_tenant_id" \
  -H "Authorization: Bearer YOUR_API_KEY"
