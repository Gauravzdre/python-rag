How to Embed This Chatbot

To add this RAG chatbot to your website, simply include this script tag in your HTML:

<script src="https://python-rag-9u2g.onrender.com/rag-widget.js" data-api-url="https://python-rag-9u2g.onrender.com/" data-tenant-id="omegagaze_online" data-title="Omegagaze AI Assistant" data-api-key="mt_107a1224597b4372"> </script>

Configuration Options:

  • data-api-url: Your RAG API server URL
  • data-tenant-id: Your tenant ID (create one in the admin panel)
  • data-api-key: Your tenant API key for authentication
  • data-title: Custom title for the chatbot
  • data-position: Position (bottom-right, bottom-left, etc.)
  • data-theme: Theme color scheme

Steps to Set Up:

  1. Start your RAG API server
  2. Create a tenant in the admin panel
  3. Upload documents for that tenant
  4. Add the script tag to your website
  5. Test the chatbot!
